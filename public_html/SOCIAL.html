:::::::::::.replaceWith (newContent) Descripción::::::::::::::::: Sustituya cada elemento del conjunto de elementos coincidentes con el nuevo contenido proporcionado y devuelva el conjunto de elementos que se eliminó. versión añadida: 1.2 .replaceWith (newContent) nuevo contenido Tipo: htmlString o Element o Array o jQuery El contenido a insertar. Puede ser una cadena HTML, un elemento DOM, una matriz de elementos DOM o un objeto jQuery. versión añadida: 1.4 .replaceWith (función) función Tipo: Función () Función que devuelve el contenido con el que reemplazar el conjunto de elementos coincidentes. El método .replaceWith() elimina el contenido del DOM e inserta el nuevo contenido en su lugar con una sola llamada. .

Docente: Ing. Freddy Leon Alumno: Wilmer Guzman Jara Ciclo:4to.

tas adicionales: El método .replaceWith() elimina todos los controladores de datos y eventos asociados con los nodos eliminados. Antes de jQuery 1.9, .replaceWith() intentaría agregar o cambiar nodos en el conjunto jQuery actual si el primer nodo del conjunto no estaba conectado a un documento y en esos casos devolver un nuevo conjunto jQuery en lugar del conjunto original. ¡El método puede o no haber devuelto un nuevo resultado dependiendo del número o la conexión de sus argumentos! A partir de jQuery 1.9, .after() , .before() y .replaceWith() devuelven siempre el conjunto original no modificado. Intentar utilizar estos métodos en un nodo sin un padre no tiene ningún efecto, es decir, ni el conjunto ni los nodos que contiene se cambian.
